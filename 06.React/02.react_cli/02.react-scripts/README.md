# react-scripts

## 目录

```md
├─ config
| ├─ env.js    加载本地.env文件环境变量配置，暴露一个方法读取环境变量（忽略）
| ├─ paths.js  包含n个本地文件/目录绝对路径（能够看得懂）
| ├─ webpack.config.js  生产/开发环境下webpack主要配置文件
| └─ webpackDevServer.config.js devServer的配置
├─ package.json
├─ scripts
| ├─ build.js  运行yarn build指令实际启动的文件
| ├─ start.js  运行yarn start指令实际启动的文件
```
